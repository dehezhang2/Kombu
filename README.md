# The Kombu Renderer 
Physically based renderer based on Nori in C++11

* Mac
 conda remove --force qt
 brew uninstall --force qt
 brew tap cartr/qt4
 brew tap-pin cartr/qt4
 brew install qt@4
 conda install -c pkgw-forge qt4
